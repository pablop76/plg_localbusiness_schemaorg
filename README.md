# Schema.org LocalBusiness dla Joomla 5

Profesjonalna wtyczka rozszerzająca system Schema.org w Joomla 5 o obsługę typu **LocalBusiness** (Lokalna Firma). Pozwala w łatwy sposób dodać bogate dane strukturalne do Twoich artykułów, co poprawia widoczność firmy w wynikach wyszukiwania Google (Local SEO).

## Główne Funkcje

- **Pełna integracja z Joomla 5:** Wykorzystuje natywny system Schema.org wprowadzony w Joomla 5.
- **Obsługa pol nazwy i tożsamości:** Nazwa prawna, alias, opis, slogany, słowa kluczowe.
- **Media:** Łatwe dodawanie logo oraz zdjęcia głównego firmy.
- **Dane kontaktowe:** Telefon, e-mail, faks.
- **Lokalizacja i Adres:** Pełny adres pocztowy, współrzędne geograficzne (Lat/Lon) oraz link do mapy.
- **Biznes i płatności:** Zakres cen, akceptowane waluty i formy płatności.
- **Godziny otwarcia:** Obsługa wielu reguł godzin otwarcia.
- **Identyfikacja prawna:** NIP (VAT ID), REGON (Tax ID), numery DUNS, LEI, NAICS oraz data założenia.
- **Media społecznościowe:** Obsługa linków `sameAs` (Facebook, Instagram, LinkedIn itp.).
- **Opcje miejsca:** Dostęp publiczny, obsługa Drive-Through, informacja o paleniu.
- **Opinie i Oceny:** Możliwość ręcznego wprowadzenia zbiorczej oceny klientów (`AggregateRating`).

## Instalacja

1. Pobierz paczkę `.zip` z zakładki [Releases](https://github.com/pablop76/plg_localbusiness_schemaorg/releases).
2. W panelu administracyjnym Joomla przejdź do **System -> Instaluj -> Rozszerzenia**.
3. Wgraj pobrany plik.
4. Przejdź do **System -> Zarządzaj -> Dodatki**, wyszukaj "Schemaorg - Lokalna Firma" i upewnij się, że dodatek jest włączony.

## Konfiguracja

1. Otwórz wybrany artykuł w edycji na zapleczu.
2. Przejdź do zakładki **Schema**.
3. Z listy rozwijanej wybierz **Lokalna Firma (LocalBusiness)**.
4. Uzupełnij dane swojej firmy w pojawiających się polach.
5. Zapisz artykuł.

## Wymagania

- Joomla 5.0 lub nowsza.
- Włączony systemowy dodatek "System - Schema.org".
