Plan dodania LocalBusiness do systemu Schema w Joomla
1. Analiza systemu Schema Joomla
przeanalizuj budowe Joomla
https://manual.joomla.org/docs/
D:\laragon\www\medycynaznatury
Sprawdzić jak działa plugin:

System – Schema.org

2. Określenie celu rozszerzenia

Celem jest dodanie nowego typu:

LocalBusiness

który będzie dostępny w:

Article Editor → zakładka Schema

tak jak inne typy.

3. Określenie pól LocalBusiness

Na podstawie dokumentacji:

https://schema.org/LocalBusiness

minimalny zestaw pól:

Business Name

Address

City

Postal Code

Country

Telephone

Website

opcjonalne:

image

priceRange

openingHours

geo

4. Integracja z edytorem artykułu

Po instalacji pluginu w edytorze artykułu pojawi się:

Schema → wybór typu

nowa opcja:

LocalBusiness

Po jej wybraniu pojawią się pola biznesowe.

5. Generowanie danych Schema

Podczas renderowania artykułu:

Joomla sprawdza typ schema artykułu

jeśli wybrano LocalBusiness

plugin generuje JSON-LD

dane są dodane do istniejącego schema strony

6. Testowanie

Sprawdzić:

poprawność JSON-LD

działanie w artykułach

kompatybilność z innymi pluginami schema

Testy:

Google Rich Results Test

Schema Validator
